首页应该展示哪些内容区块？

选出你想要的（可多选）— 这决定首页的信息架构

必

Hero 全屏水下场景

Three.js 粒子 + 气泡动画，大标题 "DIVE IN" + 入会CTA按钮。页面第一屏，震撼开场。

A

最新消息 / 新闻卡片

3-4 张横向滚动的新闻卡片，玻璃质感，hover 时光晕效果。链接到完整新闻页面。

B

即将举行的赛事 / 活动

倒计时动画 + 赛事卡片。如"2026 全国锦标赛 — 还有 47 天"。制造紧迫感和期待感。

C

数据展示区（跳动数字）

滚动触发数字动画："300+ 会员"、"25年历史"、"6个城市"、"每周训练"。增加可信度。

D

如何参与 / 游戏介绍

横向步骤动画："1. 潜水 → 2. 控球 → 3. 射门"。吸引新人入门，配视频或 GIF。

E

精彩照片墙

Masonry 瀑布流照片网格，hover 展开 + 发光边框。展示比赛、训练的真实氛围。

F

加入我们 CTA 区

全宽深海背景 + "开始你的水下旅程" 大标题 + 报名按钮。页面底部的行动号召。

多选即可，"必"是固定的。选好后告诉我。
